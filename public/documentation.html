Documentation

This is the documentation for the API of the "myFlix" application.

Introduction

This API will be dealing with requests to the database of the "myFlix" application.
It will provide data about movies, genres, directors.
Using the application user will be able to sign up. Therefore the api will also deal with user interactions with the database, e.g. updating their personal information.
